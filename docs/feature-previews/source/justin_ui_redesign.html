02

工具卡片：8 張等大，視覺主從很弱

舊版 etf-feature-grid 一律 2 欄 × min-height 82px，「核心 / 常用」chip 試圖區分但背景色差異太小，視覺上 8 張幾乎一樣重。功能性質不同（決策工具、查詢工具、行事曆、管理員）也沒分組。

04

ETF 詳細頁 hero：5 個 mini-pill 視覺平均

舊版 .etf-hero + 5 個 .mini-pill 一律一行：費率、殖利率、折溢價、規模、受益人。5 個資訊重要性其實不同，但被同等大小呈現；又混雜 tone-blue / purple / red / amber，色彩語意混亂。
